# Polaires Excel / Excel polars

Fichier Excel de modélisation de polaires des vitesses (planeur)

Méthodes :
- Méthode Cambridge
- Méthode Reichmann
- Méthode Naviter
- régression polynomiale 2nd degré
- régression polynomiale degré 4
